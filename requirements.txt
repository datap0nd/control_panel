Top 3 accounts by 2026 sales

Assume the source table is named SalesData and has these columns:
channel, customer, orgsales, seg1, category, value

In the output table, use one row per rank, where rank is 1, 2, or 3.

Formula to return the top account for the row's channel / country / seg1,
ranked by 2026 sales:

=LET(
  ch,[@channel],
  ctry,[@customer],
  seg,[@seg1],
  n,[@rank],
  accts,UNIQUE(FILTER(SalesData[orgsales],
    (SalesData[channel]=ch)*
    (SalesData[customer]=ctry)*
    (SalesData[seg1]=seg)*
    (SalesData[category]=2026)
  )),
  sales2026,SUMIFS(
    SalesData[value],
    SalesData[channel],ch,
    SalesData[customer],ctry,
    SalesData[seg1],seg,
    SalesData[category],2026,
    SalesData[orgsales],accts
  ),
  IFERROR(INDEX(SORTBY(accts,sales2026,-1),n),"")
)

Formula to fetch the value for that returned account in the row's selected year:

=SUMIFS(
  SalesData[value],
  SalesData[channel],[@channel],
  SalesData[customer],[@customer],
  SalesData[seg1],[@seg1],
  SalesData[category],[@category],
  SalesData[orgsales],[@TopAccount]
)

Important:
The ranking formula always uses category = 2026.
The value formula uses the output row's category/year.
