1. In erp_master.xlsx, create a plain worksheet with Material in cell A1 and the material codes below it.
2. If the codes are currently in a PivotTable, copy the row labels and paste them as values into this plain worksheet first.
3. Select the Material header and all material-code rows.
4. Press Ctrl+T.
5. Check My table has headers, then select OK.
6. Click anywhere inside the new table.
7. Open the Table Design tab.
8. In the Table Name box at the far left, replace the current name with erp_master and press Enter.
9. Save and close the workbook.
