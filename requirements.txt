Perfect OrgSales rank formula

Headers start in row 3, beginning in column C:
C = Channel
D = Customer / Country
E = OrgSales / Account
F = Biz Group
G = Seg1
H = Plan
I = Category / Year
J = 1 Q.
K = Total
L = Value / Sales
M = 2 Q.
N = Rank helper

Paste this in N4 and copy it down.

This formula ranks the current row's OrgSales account from E4 by
descending Value from column L.

Dynamic context from current row:
C = Channel
D = Customer / Country
F = Biz Group

Hard filters:
G / Seg1 = Sub Total
I / Category = 2026

Ignored:
H / Plan
J / 1 Q.
K / Total
M / 2 Q.

Formula for N4:

=LET(
  ch,TRIM(SUBSTITUTE($C4&"",CHAR(160)," ")),
  ctry,TRIM(SUBSTITUTE($D4&"",CHAR(160)," ")),
  acct,TRIM(SUBSTITUTE($E4&"",CHAR(160)," ")),
  biz,TRIM(SUBSTITUTE($F4&"",CHAR(160)," ")),
  chanCol,TRIM(SUBSTITUTE($C$4:$C$100000&"",CHAR(160)," ")),
  custCol,TRIM(SUBSTITUTE($D$4:$D$100000&"",CHAR(160)," ")),
  acctCol,TRIM(SUBSTITUTE($E$4:$E$100000&"",CHAR(160)," ")),
  bizCol,TRIM(SUBSTITUTE($F$4:$F$100000&"",CHAR(160)," ")),
  segCol,TRIM(SUBSTITUTE($G$4:$G$100000&"",CHAR(160)," ")),
  yearCol,TRIM(SUBSTITUTE($I$4:$I$100000&"",CHAR(160)," ")),
  valCol,$L$4:$L$100000,
  sales,IFERROR(1*SUBSTITUTE(SUBSTITUTE(SUBSTITUTE(valCol&"",",",""),"$","")," ",""),0),
  ctx,(chanCol=ch)*
      (custCol=ctry)*
      (bizCol=biz)*
      (segCol="Sub Total")*
      (yearCol="2026")*
      (acctCol<>""),
  accts,UNIQUE(FILTER(acctCol,ctx)),
  totals,BYROW(accts,LAMBDA(a,SUM(FILTER(sales,ctx*(acctCol=a),0)))),
  myTotal,XLOOKUP(acct,accts,totals),
  RANK.EQ(myTotal,totals,0)
)

Important:
There is no outer IFERROR on purpose. If it cannot calculate, Excel
should show the real error instead of silently returning blank.

How it works:
1. Build a filtered pool of rows with the same Channel, Customer, and
   Biz Group as the current row.
2. Inside that pool, keep only Seg1 = Sub Total and Category = 2026.
3. Get the unique OrgSales accounts in that pool.
4. Sum Value for each OrgSales account.
5. Find the current row's OrgSales total.
6. Return its RANK.EQ position, descending by Value.

Rank 1 = highest Value.
Rank 3 = two OrgSales accounts have higher Value.

If Excel uses semicolon separators, replace commas with semicolons.
